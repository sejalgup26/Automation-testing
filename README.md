# AI-Assisted Testing Demo Project

This project demonstrates how AI can be used in modern software testing to:
- Generate test cases from requirements
- Identify edge scenarios
- Assist in automation test creation
- Improve API test coverage

## Tech Stack
- Manual Testing + AI
- Selenium + Java + TestNG
- API Testing (Postman / Rest Assured concept)
- AI Prompt Engineering

## Key Highlights
- AI-generated test cases and edge scenarios
- Page Object Model automation framework
- API testing with negative scenarios
- Interview-ready explanations

## How AI Is Used
- Requirement → Test Case generation
- Edge case discovery
- Automation structure generation
- API test scenario generation
